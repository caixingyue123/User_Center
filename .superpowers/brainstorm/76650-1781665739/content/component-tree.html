🔄 数据流

登录流程

① LoginPage 输入用户名密码
② 调用 api/user.ts → POST /api/v1/login
③ 后端返回 {user, token}
④ stores/user.ts 存 token + userInfo
⑤ utils/token.ts 写 localStorage
⑥ router 跳转 /dashboard

API 请求统一链路

组件调用 → api/user.ts
→ request.ts axios 实例
→ 请求拦截器：自动带 Authorization
→ Vite proxy：/api → :8080
→ 响应拦截器：统一处理 code≠0
→ 401 → 清除 token → 跳登录
